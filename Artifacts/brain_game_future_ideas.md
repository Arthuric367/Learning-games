## Pending Game Ideas

1. Color Route Finder
- Goal: Follow the requested color path through many crossings.
- Core skill: Visual discrimination and selective attention.
- Difficulty: Similar shades, denser crossings, longer route.

2. Shape Tunnel Track
- Goal: Follow a shape rule (e.g., circle path first, then triangle path).
- Core skill: Rule switching and cognitive flexibility.
- Difficulty: Mid-round rule changes and more distractor paths.

3. Dot-to-Dot Patrol
- Goal: Connect dots in sequence while ignoring decoys.
- Core skill: Sequencing and sustained focus.
- Difficulty: Reverse order, skip counting, and fake near matches.

4. Calm Breath Path
- Goal: Path advances only when input rhythm matches calm breathing pace.
- Core skill: Self-regulation and paced motor control.
- Difficulty: Longer sections and tighter rhythm tolerance.

5. Spot the Safe Exit
- Goal: Pick the path that reaches exit while avoiding hazards.
- Core skill: Planning and decision making in complex scenes.
- Difficulty: Multi-step objectives (collect key, then exit).

6. Hidden Path Reveal
- Goal: Reveal limited tiles and infer the safe path.
- Core skill: Pattern inference and delayed response.
- Difficulty: Fewer reveals and larger boards.

7. Balloon Breathing Race
- Goal: Control balloon movement with slow inhale/exhale rhythm.
- Core skill: Breath control and emotional reset.
- Difficulty: Moving obstacles and longer route.

8. Emotion Detective
- Goal: Read short scenario and choose feeling + best calm action.
- Core skill: Emotional labeling and coping response selection.
- Difficulty: More subtle social situations.

9. Freeze and Flow
- Goal: Move with music cues and freeze on stop signals.
- Core skill: Inhibition and attention switching.
- Difficulty: Faster cue shifts and mixed cue types.

10. Build-After-Breathe
- Goal: Earn puzzle pieces only after completing calm breath steps.
- Core skill: Delay tolerance and self-regulation before reward.
- Difficulty: Longer chains before reward unlock.

## Notes for Next Implementation Batch
- Keep rounds short (6-8) to reduce fatigue.
- Add Easy/Advanced toggles for consistency with current games.
- Use calm-retry feedback instead of strong penalties.
- Preserve touch-friendly controls for mobile first use.
